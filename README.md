## Description

API that uses chatGPT and elevenlabs to generate the text and transcribe it to audio.
<br><br>
Using JWT authentication.

## Installation

```bash
$ npm install
```

## Docker

```bash
# development
sudo docker-compose up --build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
